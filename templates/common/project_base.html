Description

Small demo developed as three milestones coursework at uni. It's inspired by the Super Mario games but it has a slightly different way to succed the levels: gathering enough roses to give to the princess in order to ulock the next level.

The game provides a simple but efficient GUI that lets navigate between levels and replay them as well.
Elementary AI has been included in the game to raise a little bit the overall level of fun.

Details

  • Java 8 OOP
  • JBox2D Physics Engine
  • UI
  • HUD
